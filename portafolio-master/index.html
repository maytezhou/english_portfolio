Portfolio

The following Projects were built as part of Laboratoria- Peruvian Coding Bootcamp

Portfolio

Social Networking app

Breath Life: It is a social network in Spanish. In this platform, the user can post pictures, share advice or tips about what they do in their everyday life to stay healthy.

Technologies used: #Using firebase as database # MVC Model View Controller #Single Page Application #Vanilla Javascript #Jest

Md-Links Library

Md-Links: It is a library which is able to analyze Markdown files and the output returns data about the links found in each file.

Technologies used: #Jest #Node Js #Node modules #Babel #Asynchronous Javascript . The following mocks were implemente such as: module #mocks-fs and #fetch-mock

Data Lovers

Data Lovers: This platform displays data from the Development Indicators of Peru provided by the World Bank. The user can filter the data by any category he selects. It is only available in Spanish

:Technologies used #Vanilla Javascript #CSS3 #HTML5 #Jest

Skills learned: Vanilla Javascript, arrays, objects, template strings, pure functions, higher order functions: map, filer, sort, reduce

Burguer Queen

Burguer Queen: This platform seeks to contribute to the easy and effective management of any type of restaurant. It has a Waiter section, where you can write down the orders of each client and calculate the total cost, while in the Cooker section, you can access a list of orders. It is only available in Spanish.

Technologies used: #React #Hooks #Bootstrap
